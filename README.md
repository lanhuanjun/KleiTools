# KleiTools
Convert kex files and png files to each other in Don't Starve.
# Usage
```shell
kleitool.exe -i xxx[file|dir]  -o xxx[file|dir] -t [0:DTX1, 1:DTX3 2:DTX5 4:ARGB 5:RGB 18:ETC2_EAC] -p [dir, output to png]
```
